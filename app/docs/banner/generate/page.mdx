# Generate Banner

<span className="inline-block px-2 py-0.5 rounded text-xs font-bold bg-emerald-100 text-emerald-700 mr-2">POST</span>
`/api/v1/banner/generate`

Generate a professional AI-powered banner ad from a single API call. The AI creates a creative strategy (headline, subtext, CTA, visual concept) and then generates the banner image.

Returns a `jobId` immediately — the banner is generated in the background (~30–90 seconds). Use the [status endpoint](/docs/banner/status) or a webhook to get the result.

## Generation Pipeline

The banner goes through three phases:

| Phase | Description | Duration |
|-------|-------------|----------|
| **strategy** | AI generates creative brief (headline, subtext, CTA, visual concept, psychology trigger) | ~5-10s |
| **image** | Banner image generated based on the strategy | ~30-60s |
| **critique** | Hebrew text quality check + auto-retry if needed (Hebrew only) | ~10-15s |

## Error Responses

| Status | Description |
|--------|-------------|
| `400` | Missing required fields (`businessName`, `businessDescription`) |
| `401` | Invalid or missing API key |
| `402` | Insufficient tokens |
| `429` | Rate limit exceeded |

## Using Custom Copy

You can override the AI-generated copy with your own text:

<CodeBlock
  language="json"
  code={`{
  "businessName": "Café Sol",
  "businessDescription": "Artisan coffee shop",
  "customCopy": {
    "headline": "הטעם שמתחיל את הבוקר",
    "subtext": "קפה טרי כל יום מ-6:00",
    "cta": "הזמינו עכשיו"
  },
  "aspectRatio": "1:1"
}`}
/>

<Callout type="info">
  When custom copy is provided, the AI skips the strategy phase and goes directly to image generation using your text.
</Callout>

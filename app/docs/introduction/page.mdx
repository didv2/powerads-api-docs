# Welcome to PowerAds API

PowerAds lets you generate professional AI-powered video ads through a simple REST API. Describe your business, choose a style, and get a production-ready video — all via a single API call.

## Base URL

All API requests are made to:

<CodeBlock code="https://video-api.powerads.ai" language="text" />

## Capabilities

- **Video Generation** — Create AI-powered video ads with scripts, voiceover, music, captions, and AI-generated visuals
- **Multiple Styles** — UGC, cinematic, product-focused, and more
- **AI Presenters** — Use custom or catalog video models as on-screen talent
- **Webhook Notifications** — Get notified when your video is ready
- **Multi-language Support** — Generate ads in Hebrew, English, and more

<Callout type="info" title="More coming soon">
  We're actively expanding the API. Image generation, A/B testing, and campaign management endpoints are on the roadmap.
</Callout>

## Quick Start

Here's a minimal example to generate a video ad:

<CodeBlock
  language="bash"
  title="Generate a video"
  code={`curl -X POST https://video-api.powerads.ai/api/v1/video/generate \\
  -H "Content-Type: application/json" \\
  -H "x-api-key: pk_live_your_api_key_here" \\
  -d '{
    "businessName": "Café Sol",
    "businessDescription": "Artisan coffee shop in Tel Aviv",
    "language": "he",
    "adStyle": "ugc"
  }'`}
/>

**Response:**

<CodeBlock
  language="json"
  title="200 OK"
  code={`{
  "jobId": "job_abc123def456",
  "status": "processing",
  "message": "Video generation started. Use the status endpoint to track progress."
}`}
/>

Then poll the status endpoint or use a webhook to know when it's done. That's it — you're generating video ads with AI.

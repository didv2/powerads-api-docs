# Welcome to PowerAds API

PowerAds lets you generate professional AI-powered marketing assets through a simple REST API. Describe your business, choose a style, and get production-ready content — all via a single API call.

## Base URL

All API requests go through our central gateway:

<CodeBlock code="https://api.powerads.ai" language="text" />

The gateway handles authentication, rate limiting, and routes your requests to the appropriate service.

## Available APIs

| API | Prefix | Status |
|-----|--------|--------|
| **Video Generation** | `/api/v1/video/*` | ✅ Available |
| **Banner Generation** | `/api/v1/banner/*` | ✅ Available |
| Landing Pages | `/api/v1/lp/*` | 🔜 Coming soon |
| Social Posts | `/api/v1/social/*` | 🔜 Coming soon |

## Capabilities

- **Video Generation** — Create AI-powered video ads with scripts, voiceover, music, captions, and AI-generated visuals
- **Banner Generation** — Create professional banner ads with AI-generated creative strategy, copy, and visuals
- **Multiple Styles** — UGC, cinematic, product-focused, and more
- **AI Presenters** — Use custom or catalog video models as on-screen talent
- **Webhook Notifications** — Get notified when your content is ready
- **Multi-language Support** — Generate content in Hebrew, English, and more

## Quick Start

Here's a minimal example to generate a video ad:

<CodeBlock
  language="bash"
  title="Generate a video"
  code={`curl -X POST https://api.powerads.ai/api/v1/video/generate \\
  -H "Content-Type: application/json" \\
  -H "x-api-key: pk_live_your_api_key_here" \\
  -d '{
    "businessName": "Café Sol",
    "businessDescription": "Artisan coffee shop in Tel Aviv",
    "language": "he",
    "adStyle": "ugc"
  }'`}
/>

**Response:**

<CodeBlock
  language="json"
  title="200 OK"
  code={`{
  "jobId": "pipe_abc123def456",
  "status": "processing",
  "message": "Video generation started. Use the status endpoint to track progress."
}`}
/>

Then poll the status endpoint or use a webhook to know when it's done. That's it — you're generating video ads with AI.

## How It Works

```
Your App → api.powerads.ai → Video Service → Video Ready → Webhook
              (gateway)         (pipeline)
```

1. **Authenticate** with your personal API key (`pk_live_...`)
2. **Send a request** describing your business and preferences
3. **Get a job ID** immediately
4. **Track progress** via status endpoint or receive a webhook when done
5. **Download** your finished video
